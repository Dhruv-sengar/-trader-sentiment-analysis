# Trader Performance vs Market Sentiment

This project analyzes how Bitcoin market sentiment (Fear/Greed) relates to trader behavior and performance on Hyperliquid. 

## Project Structure
- `/data`: Contains the raw CSV datasets (`fear_greed_index.csv`, `historical_data.csv`).
- `/notebooks`: Contains the primary deliverable Jupyter Notebook (`main.ipynb`).
- `/outputs`: Contains saved charts and summary tables generated by the notebook.
- `/src`: Contains helper scripts (e.g., notebook generation script).

## Setup & Execution
1. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Ensure the two CSV files are placed in the `/data` directory:
   - `fear_greed_index.csv`
   - `historical_data.csv`
3. Launch Jupyter Notebook and run the main analysis:
   ```bash
   jupyter notebook notebooks/main.ipynb
   ```
   Execute all cells in `main.ipynb` to generate the insights, charts, and tables. All generated artifacts will be saved automatically into the `/outputs` folder.
